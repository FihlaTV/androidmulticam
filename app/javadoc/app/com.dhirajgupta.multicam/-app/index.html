App

class App : Application

Standard Android App subclass that is useful for most Apps. By providing the shared instance of the App object, we are able to access the Application Context anywhere that it is useful. In this particular project we're using Application Context to ask MediaScannerConnection to scan our newly saved images so that it appears in the user's Photo Gallery application.
